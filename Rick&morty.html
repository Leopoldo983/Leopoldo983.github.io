Rick & Morty App

Project on the television series Rick and Morty using JavaScript in greater proportion, making use of the API available on the web to manipulate dynamically this data and in turn modify the Dom

The function of this project is to visualize relevant information about the characters of the rick and Morty television series, this information is collected and used thanks to the API available on the web, through the manipulation of the dom with JavaScript in which containers are created of the information used to show directly to the user and depending on the interactions, you will be able to observe information of different characters of the Serie
