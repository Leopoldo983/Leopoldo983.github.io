Notes App

Notes app created in React, controlling the state of components making changes to these according to user interactions, saving the data using as a tool the LocalStorage

In this project an app was created with the React library in which some components were created for the operation of the application, concepts such as the state of components were applied to control how the components are updated according to the interactions that users will perform, the app has different functions such as creating notes, deleting them and searching for them, it has personalized storage thanks to the use of localStorage which allows the data of the app not to be deleted when creating the personalized notes
