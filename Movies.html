Movies Website

Project of movies using JavaScript in greater proportion, making calls to the API available on the web through Axios, along with tools for manipulating the dom.

In this project it is intended to offer important and updated information on movies that have already been released, in each of these we deliver data such as: popularity, category, and score have given, to do so I used the Axios tool to make calls to the API, and through javaScript use this data and inject it into the dom employing dom manipulation tools, together with a local hash navigation system which allows navigate in different parts of the page using javascript as much as possible
